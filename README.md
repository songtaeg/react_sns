🖍 기능
    
1. 로그인
   + 회원 가입

2. 회원 가입
   + 아이디 중복 체크
   + 비밀번호 / 비밀번호 확인

3. 피드
   + 피드 누르면 상세 내용 보여줌
   + 본인 피드 삭제 가능
   + 댓글 기능 (자신 댓글 삭제 가능)

4. 등록
   + 제목,내용, 사진(선택사항) 등록 
     
5. 마이페이지
   + 본인 소개(구현 X) 디폴트
